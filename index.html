Search tips

You can search by property by prefixing your query with "property:"

  • username:john
  • url:https://github.com

You can list trashed entries by using "trashed:" as query.

  • trashed:

Query are tested as regular expressions.

  • er$ — matches entries with titles that have "er" at the end
  • url:com$ — matches entries with urls that end with "com"
  • password:^asdf — matches entries with passwords that start with "asdf"
